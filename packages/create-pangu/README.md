# `@opd/create-pangu`

> scaffold tool for pangu

## Usage

### Use `npm create` or `npm init` (**recommend**)

```bash
npm create pangu demo
# or
npm init pangu demo
```

### Use `npx`

```bash
npx @opd/create-pangu demo
```

### Use global package (**NOT** recommend)

```bash
npm i @opd/create-pangu -g
create-pangu demo
```
